.. sq:function:: function uploadFile({token, filepath, filetype, partSize, sendUrl, saveUrl, sentPartEvent, savedFileEvent})

  :param token: jwt token to use for authentication
  :type token: string

  :param filepath: path to file to upload
  :type filepath: string

  :param filetype: meta info about file, will be passed in save file request
  :type filetype: table

  :param partSize: (optional), size of each part in bytes, 1 MB by default
  :type partSize: integer

  :param sendUrl: url to send parts to
  :type sendUrl: string

  :param saveUrl: url to call to save file after all parts are sent
  :type saveUrl: string

  :param sentPartEvent: (optional) : event id to send into eventbus after each part is sent
  :type sentPartEvent: string

  :param savedFileEvent: (optional) : event id to send into eventbus after file is saved
  :type savedFileEvent: string

  ``Function is kwarged - arguments passed in a table``




response for sentPartEvent in eventbus data or in callback will be table


.. code-block:: 
    :caption: response
    

    {
      "http_status" : integer (SUCCESS, FAILED, ABORTED)
      "http_code" : integer
      "uuid" : string
      // if http_status is SUCCESS
      "status" : string, optional
      "msg" : string, optional
      "state" : string ("CONTINUE" or "DONE")
      "received" : integer, optional, how many parts was received. Present if state is CONTINUE
    }

response for savedFileEvent in eventbus data or in callback will be table


.. code-block:: 
    :caption: response
    

    {
      "http_status" : integer (SUCCESS, FAILED, ABORTED)
      "http_code" : integer
      "uuid" : string
      // if http_status is SUCCESS
      "status" : string, optional
      "msg" : string, optional
      "hash" : string, optional, hash of the saved file
      "uri" : string, optional, uri of the saved file
      "new" : bool, optional, true if file was newly saved, false if file with such hash already exists
    }
