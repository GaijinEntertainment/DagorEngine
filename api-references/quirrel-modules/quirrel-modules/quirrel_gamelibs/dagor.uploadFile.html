dagor.uploadFile

module ‘dagor.uploadFile’

Source files:

prog/gameLibs/quirrel/fileUpload/fileUpload.cpp

Usage:

let { uploadFile } = require("dagor.uploadFile")

or:

from "dagor.uploadFile" import *

//'dagor.uploadFile' exports:
{
  function uploadFile({token, filepath, filetype, partSize, sendUrl, saveUrl, sentPartEvent, savedFileEvent}): table -> _undocumented_
}
function uploadFile({token, filepath, filetype, partSize, sendUrl, saveUrl, sentPartEvent, savedFileEvent})
Arguments:

  • token (string()) – jwt token to use for authentication

  • filepath (string()) – path to file to upload

  • filetype (table()) – meta info about file, will be passed in save file request

  • partSize (integer()) – (optional), size of each part in bytes, 1 MB by default

  • sendUrl (string()) – url to send parts to

  • saveUrl (string()) – url to call to save file after all parts are sent

  • sentPartEvent (string()) – (optional) : event id to send into eventbus after each part is sent

  • savedFileEvent (string()) – (optional) : event id to send into eventbus after file is saved

Function is kwarged - arguments passed in a table

response for sentPartEvent in eventbus data or in callback will be table

response
{
  "http_status" : integer (SUCCESS, FAILED, ABORTED)
  "http_code" : integer
  "uuid" : string
  // if http_status is SUCCESS
  "status" : string, optional
  "msg" : string, optional
  "state" : string ("CONTINUE" or "DONE")
  "received" : integer, optional, how many parts was received. Present if state is CONTINUE
}

response for savedFileEvent in eventbus data or in callback will be table

response
{
  "http_status" : integer (SUCCESS, FAILED, ABORTED)
  "http_code" : integer
  "uuid" : string
  // if http_status is SUCCESS
  "status" : string, optional
  "msg" : string, optional
  "hash" : string, optional, hash of the saved file
  "uri" : string, optional, uri of the saved file
  "new" : bool, optional, true if file was newly saved, false if file with such hash already exists
}
