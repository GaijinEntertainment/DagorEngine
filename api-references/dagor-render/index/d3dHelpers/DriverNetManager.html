class DriverNetManager
#include <dag_driverNetManager.h>

The DriverNetManager interface.

This class is responsible for providing network related functionality for d3d drivers. Some drivers need to send data from clients for debugging and infrastructure purposes. Methods of the class are very specific currently because we have a very strict set of use cases that we want to cover.

virtual void sendPsoCacheBlkSync(const DataBlock &cache_blk) = 0

Send a cache block to the server.

This method is used to send a cache block to the server. A call of this method blocks the thread execution until the data is sent or the request is timed out. The blocking behavior is necessarty to ensure that all the data is send and it doesn’t affect the players experience since we are sending data only when the driver shuts down.

Parameters:

cache_blk – The cache data block to send

virtual void sendHttpEventLog(const char *type, const void *data, uint32_t size, Json::Value *meta) = 0

Send an event log to the server.
