# Shader: rendinst_deferred_modulate2x_decal

## Overview

It's a deferred version of
[rendinst_modulate2x_diffuse_decal](rendinst_modulate2x_diffuse_decal.md).

It is visible only when have other geometry behind and can be used in
[composites](../../all-about-blk/composit_blk.md), unlike the base version
(which affects only geometry of the same `.dag` without any depth checks).
