OverviewÔÉÅ

This shader is designed for prefabs and render instances with additional detailing, including a normal map.

In 3ds Max, the shader‚Äôs class name is simple_aces_detailed. It uses the following texture slots:

  • Slot 4 ‚Äì Normal map for the detail texture

  • Slot 8 ‚Äì Detail texture, with tiling controlled by the parameters detail_scale_u=8 and detail_scale_v=8

  tex2:t="./karelia_cliff_a_b_tex_n.tif"

  • class:t="simple_aces_detailed" ‚Äì The shader class being used for the material.

  • tex16support:b=yes ‚Äì Enables support for 16-bit textures.

  • twosided:i=0 ‚Äì Determines whether the material is rendered double-sided. In this case, 0 means it‚Äôs single-sided.

  • spec:ip3=255, 255, 255 ‚Äì Specular color (RGB: 255, 255, 255).

  • emis:ip3=0, 0, 0 ‚Äì Emission color (RGB: 0, 0, 0), meaning no self-illumination.
